## How to Flash

### Option 1: Web Flasher (Easiest)
Visit: https://YOUR_USERNAME.github.io/esp32-captive-portal/flasher/

### Option 2: esptool.py
```bash
# Install esptool
pip install esptool

# Flash (replace COM3 with your port)
esptool.py --chip esp32s3 --port COM3 write_flash 0x0 EvilPortal_v1.1.0_esp32s3.bin
```

### Option 3: Arduino IDE
1. Go to **Tools > ESP32 Sketch Data Upload** (if you have the plugin)
2. Or use **Sketch > Upload** after opening the source

## Generating New Binaries

See [docs/BUILDING.md](../docs/BUILDING.md) for compilation instructions.

Binaries are automatically generated by GitHub Actions when you push a version tag:
```bash
git tag v1.1.0
git push origin v1.1.0
```
